Clojure Crud
======

Simple Crud in Clojure + AngularJS + MongoDB


Install leiningen
-------------------------------------------------------------
$ brew install leiningen

Install mongodb
--------------------------------------------------------------
$ sudo npm install -g mongodb<br>

Install Bower
--------------------------------------------------------------
$ sudo npm install -g bower

Start the app:
--------------------------------------------------------------

$ cd /pismo_products_clojure/<br>
$ leiningen ring server

--------------------------------------------------------------
